#  Browser Extension

The browser extension acts as the first layer of protection. It continuously monitors supported AI platforms and intercepts prompts before they are submitted. Every prompt is securely forwarded to the backend for analysis, allowing ShadowAI to identify potential risks without interrupting the user's workflow.

---

#  AI Risk Engine

The AI Risk Engine is responsible for analyzing every prompt and determining whether it contains sensitive information. Instead of relying on a single detection technique, it combines multiple approaches to improve accuracy and reduce false positives.

The engine is capable of detecting:

- API keys and authentication tokens
- Passwords and credentials
- Proprietary source code
- Personally identifiable information (PII)
- Internal business documents

After the analysis, an explainable risk score is generated along with the reasons behind the decision.

---

#  Policy Engine

The Policy Engine applies organization-specific security policies to every analyzed prompt. Based on the calculated risk score and configured rules, it decides whether the prompt should be allowed, warned, or blocked.

Organizations can customize policies according to their security requirements, departments, or compliance standards while maintaining a consistent experience for all users.

---

#  Admin Dashboard

The Admin Dashboard provides a centralized interface for monitoring AI usage across the organization. It enables administrators to review security incidents, analyze usage trends, configure policies, and monitor high-risk activities from a single location.

### Dashboard Features

- AI usage analytics
- Policy management
- Audit log viewer
- Risk reports
- User activity monitoring

---

# Audit & Monitoring

Every security decision is recorded to create a complete audit trail. These logs help administrators understand how AI tools are being used, investigate incidents, and generate reports for compliance and internal reviews.

Each audit record includes important details such as the timestamp, calculated risk level, applied policy, and final decision.

---

#  User Experience

ShadowAI is designed to protect users without disrupting productivity. Instead of simply blocking prompts, the system provides clear explanations whenever a potential risk is detected, allowing users to understand and resolve issues before resubmitting.

Users receive immediate feedback while continuing to work with their preferred AI tools.
