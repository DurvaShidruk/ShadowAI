# 1 General

### 1.1 What is ShadowAI?

ShadowAI is an AI-native security platform that helps organizations prevent sensitive information from being accidentally shared with Generative AI tools. It analyzes prompts before they are submitted and takes the appropriate action based on organizational security policies.

---

### 1.2 Why was ShadowAI built?

As AI tools become part of everyday work, employees often paste source code, credentials, customer information, or confidential documents into public AI platforms. ShadowAI was created to reduce this risk without preventing people from using AI productively.

---

### 1.3 How is ShadowAI different from traditional DLP solutions?

Traditional Data Loss Prevention (DLP) tools primarily focus on emails, file transfers, and network traffic. ShadowAI is designed specifically for AI prompts, allowing it to understand and analyze interactions with Generative AI platforms before data is shared.

---

### 1.4 Does ShadowAI block every prompt?

No. ShadowAI uses a risk-based approach. Depending on the analysis and organizational policies, a prompt may be allowed, warned, or blocked. This helps maintain productivity while protecting sensitive information.

---

# 2 Security

### 2.1 Does ShadowAI store every prompt?

Organizations can configure what information is stored. By default, audit logs focus on security events and metadata required for monitoring, reporting, and compliance rather than retaining unnecessary prompt content.

---

### 2.2 How are users notified about risky prompts?

If a prompt contains potentially sensitive information, the browser extension immediately displays a warning explaining the detected risk. For high-risk prompts, the system can prevent submission based on the configured policy.

---

### 2.3 Can organizations customize security policies?

Yes. Administrators can define their own policies, risk thresholds, and security rules to match their internal requirements and compliance standards.

---

# 3 Deployment

### 3.1 Which AI platforms are supported?

ShadowAI currently supports browser-based AI platforms such as ChatGPT, Google Gemini, and GitHub Copilot Web. The platform is designed to support additional AI services through future integrations.

---

### 3.2 Can ShadowAI be integrated into existing enterprise environments?

Yes. The modular architecture makes it suitable for integration with enterprise authentication systems, dashboards, and future governance platforms without requiring major workflow changes.

---

# 4 Future Plans

### 4.1 What features are planned for future releases?

Future enhancements include support for additional AI platforms, semantic prompt analysis using large language models, integrations with collaboration tools such as Slack and Microsoft Teams, IDE extensions, enterprise identity management, and advanced analytics.

---

### 4.2 Is ShadowAI intended only for large enterprises?

No. While it is designed with enterprise security in mind, the platform can also benefit startups, educational institutions, and small businesses that want to adopt AI responsibly.
